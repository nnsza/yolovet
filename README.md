## 产品特色

- **画布优先的标注体验**：大图浏览、**Ctrl + 滚轮**缩放、**空格 + 左键**拖动画布平移；顶栏提供**全图缩略图**与当前视野框，可在红框上拖拽、滚轮缩放视野。
- **多种框选方式**：**R** 两点矩形；**E** 柔性外接（轨迹一笔成框）；**F** 连续柔性外接（自相交成环等规则），适合复杂轮廓的快速外接。
- **数据集与标签**：选择图片目录后列出工作集；标签为与图片同名的 **YOLO 格式 `.txt`**；根目录 **`class_log.txt`** 维护类别名与索引顺序，与训练数据一致。
- **类别管理**：默认类别、排序、颜色、删除整类（可撤销）等，集中在一处维护。
- **ONNX 辅助检测**：可载入 Ultralytics 导出的 **ONNX**（建议 `nms=True, batch=1`），虚线预览检测框；支持按类别筛选显示、调节置信度、**采纳当前图**或**全局批量采纳**（与已有框 IoU 过高则跳过，避免重复）。
- **训练环境与灵活性**：软件**自带 `yolo_cpu` 训练环境**（开箱即可用于 CPU 侧训练，无需先自行搭一套）；同时支持在界面中**自由选定本机已有的 Conda 环境**，便于使用你已安装好的 PyTorch / CUDA 或其它依赖。
- **训练衔接**：在界面中配置模型与 epoch 等参数，生成训练包并启动 **Ultralytics YOLO** 训练流程；训练日志在底部面板查看。
- **撤销与效率**：**Ctrl+Z** 撤销；**A / D** 切换上一张 / 下一张图片；选中框可用滚轮快速切换类别（未按 Ctrl 时）。

## 使用方式

### 运行程序

1. 在本机编译：在项目目录执行 `cargo build --release`，可执行文件位于 `target/release/`（Windows 下一般为 `yolo_trainer_gui.exe`）。
2. 双击运行生成的可执行文件即可启动图形界面。

（若你使用其他分发方式，将程序与依赖一并放置后同样直接运行主程序即可。）

### 推荐工作流

1. **选择图片目录**  
   在侧栏「数据集」中选择作为工作区的文件夹（当前层级图片；标签规则以界面说明为准）。

2. **维护类别**  
   在「类别管理」中编辑类别名、顺序与颜色；默认类别会影响**新框**的默认类别。

3. **标注**  
   - 用 **R** 进入矩形两点模式，在画布上拉框；  
   - 或用 **E / F** 按提示完成柔性外接类标注；  
   - 输入或选择类别名并保存为当前图的 `.txt`。

4. **（可选）辅助检测**  
   载入 ONNX 模型，等待推理完成后，对当前图或全数据集执行「采纳」，将虚线预测转为正式标注（注意与现有框的 IoU 去重规则）。

5. **训练**  
   可使用自带的 **`yolo_cpu` 环境**直接开训，或从列表中**自选 Conda 环境**（例如已配置 GPU 的环境）；再选择模型与 epoch 等，按界面提示生成训练包并启动训练；在底部查看日志直至完成，按需在输出目录获取权重与导出模型。

### 部分快捷键（完整列表见软件内「查看完整快捷键」）

| 操作           | 说明（摘要）        |
|----------------|---------------------|
| Ctrl + 滚轮    | 缩放画布            |
| 空格 + 左键拖动 | 平移视野            |
| R / E / F      | 矩形 / 柔性外接模式 |
| A / D          | 上一张 / 下一张图   |
| Del / Q        | 删除                |
| Esc            | 取消当前操作        |
| Ctrl + Z       | 撤销                |

## 说明与反馈

- 具体选项名称、路径与按钮以你当前软件版本界面为准。  
- **ONNX 辅助推理**使用程序内置逻辑，一般无需额外配置。  
- **训练**：若使用自带 **`yolo_cpu`**，按界面说明即可；若**自选 Conda 环境**，请确保该环境中已安装 **Ultralytics** 等训练所需依赖，并留意界面报错与环境提示。
